# CRM Frontend

Customer Relationship Management System - Frontend

## Features
- Vue 3 with TypeScript
- Vue Router for navigation
- Pinia for state management
- OAuth2 integration
- Responsive design

## Development

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

The build output will be placed in `../src/main/resources/static` for integration with Spring Boot.
